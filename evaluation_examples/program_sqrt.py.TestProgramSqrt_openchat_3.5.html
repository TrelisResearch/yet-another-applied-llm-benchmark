def my_sqrt(x, tolerance=1e-10, max_iterations=100):
    if x < 0:
        raise ValueError("Input must be a non-negative number")

    guess = x / 2.0
    for _ in range(max_iterations):
        new_guess = (guess + (x / guess)) / 2.0
        if abs(new_guess - guess) < tolerance:
            break
        guess = new_guess

    return new_guess
This function takes a number `x` as input and returns its square root. The `tolerance` parameter is the desired precision of the result, and `max_iterations` is the maximum number of iterations allowed. The function starts with an initial guess of half the input number and iteratively refines the guess using the Newton-Raphson method until the desired precision is reached or the maximum number of iterations is exceeded.

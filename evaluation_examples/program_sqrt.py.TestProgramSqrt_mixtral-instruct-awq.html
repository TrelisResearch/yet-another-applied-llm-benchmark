def my_sqrt(number, precision=1e-15):
    """
    Computes the square root of a number using Newton's method.

    :param number: The number to find the square root of.
    :param precision: The desired level of precision (default: 1e-15).
    :return: The square root of the number.
    """

    if number < 0:
        raise ValueError("Cannot compute the square root of a negative number")

    guess = number / 2.0
    while True:
        better_guess = (guess + number / guess) / 2.0
        if abs(guess - better_guess) < precision:
            return better_guess
        guess = better_guess
This function takes a number and an optional precision parameter. It starts with an initial guess of the square root (number/2) and then iteratively refines the guess using the formula: `new_guess = (guess + number/guess) / 2` The loop continues until the difference between the new guess and the previous guess is smaller than the specified precision. This ensures a highly accurate result.

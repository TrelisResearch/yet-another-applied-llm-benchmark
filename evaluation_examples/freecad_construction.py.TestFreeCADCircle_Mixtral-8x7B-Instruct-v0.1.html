In FreeCAD, you can create a circle for reference purposes without it being a full solid object that you can extrude or perform other solid operations on. To do this, you can create a "Datum Circle" which is a type of construction geometry. Here are the steps to create a Datum Circle in FreeCAD: 1. Open FreeCAD and create a new document. 2. Go to the "Part" workbench by clicking on its icon or selecting it from the "Workbench" menu. 3. Click on the "Create shape" button in the toolbar or select "Create shape" from the "Part" menu. 4. In the "Create shape" dialog box, select "Circle" from the list of shapes. 5. Set the radius of the circle to the desired value. 6. Instead of clicking "OK" to create the circle, click on the "More" button at the bottom of the dialog box. 7. In the "Placement" section, check the box next to "Construction" to make the circle a construction geometry. 8. Click "OK" to create the Datum Circle. The circle will now appear in your 3D view as a dashed line, indicating that it is a construction geometry. You can use this circle as a reference to line up other lines or shapes, but it will not be included as part of any solid object that you create.

2. Load the pre-trained model using `AutoModel.from_pretrained`: 
1
model = AutoModel.from_pretrained("pretrained_model_name")
Replace `"pretrained_model_name"` with the name of the pre-trained model you want to use. 3. Add a language model head on top of the pre-trained model: 
1
2

# Add a linear layer on top of the pre-trained model
model.resize_token_embeddings(len(tokenizer))
model.lm_head = nn.Linear(model.config.hidden_size, len(tokenizer))
Here, `tokenizer` is an instance of `AutoTokenizer` that you should load before loading the model. The `resize_token_embeddings` method is used to adjust the size of the token embeddings to match the size of the tokenizer vocabulary. The `nn.Linear` layer is used to create a language model head that maps the hidden size of the pre-trained model to the size of the tokenizer vocabulary. 4. Initialize the weights of the language model head:

# Initialize the weights of the language model head
model.lm_head.weight = model.transformer.dense.weight

Here, the weights and bias of the language model head are initialized with the weights and bias of the last layer of the pre-trained model. 5. Fine-tune the model with your downstream task.

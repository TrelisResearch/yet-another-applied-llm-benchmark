import chess.pgn


def print_all_prefixes(game: chess.pgn.Game) -> None:
    """
    Print the PGN notation for all prefix subgames of the provided chess.pgn.Game.

    :param game: A chess.pgn.Game
    """
    for i in range(1, 1 + len(list(game.mainline()))):
        prefix = chess.pgn.Game().variance(game.before(i), game.mainline(start=i - 1, end=i))
        # We use chess.pgn.Game.as_string with default arguments,
        # which includes the PGN headers and the [Event "..."] line.
        print(prefix.as_string(headers=game.headers, initial_bracketing=True))
This function works by iterating over the mainline of the provided `chess.pgn.Game` and creating prefix subgames of varying sizes. For each subgame, it constructs a new `chess.pgn.Game()` instance with the corresponding prefix and prints the result of `as_string` for that instance.
